NetNote!


NetNote! is a simple graphical notepad writen in Java using JavaFX library. It can edit files while displaying them in one of 4 diffrent fonts. Screenshot of the program is avaliable here

Version: 2.0 | 1.0 -> 2.0 (new responsive design, changing fonts)

Year of release: 2021

Platforms: GNU/Linux, Windows (NOTE: not translated to english yet)

Programming languages: Java 16

License: Apache 2.0
