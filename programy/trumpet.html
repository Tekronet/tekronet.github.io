Trumpet


Trumpet is a simple audio player written in Java using JavaFX framework. It can play MP3, WAV and MP4 (audio only) files. It has basic features such as playback control, progress slider, volume & speed control and playlist. Screenshot of the program is avaliable here




Program info

Version: 2.0 | 1.0 -> 2.0 (improved interface, added playlist, faster startup time)

Year of release: 2022

Platforms: GNU/Linux, Windows, MacOS (need to compile from source)

Programming languages: Java 17

License: Apache 2.0

Running Trumpet

On Linux you can download Trumpet as AppImage. You just need to give it executable permisions using chmod or your graphical file manager. After that you can just run Trumpet.


For Windows there is a graphical installer that will install Trumpet on your computer.

NOTE: You need to run installer as administrator.


On MacOS you need to compile Trumpet from source code. Instructions on how to do this are on Trumpet Github page.
